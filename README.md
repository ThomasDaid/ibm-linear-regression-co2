# IBM-linear-regression-co2

This project demonstrates how Machine Learning can be used to predict vehicle CO₂ emissions based on engine characteristics.

The notebook was developed as part of a hands-on lab from the **IBM Data Science Professional Certificate**.

## Problem

Vehicle emissions are a major contributor to greenhouse gases.  
The objective of this project is to build a **Simple Linear Regression model** capable of predicting CO₂ emissions based on engine size.

## Dataset

Fuel Consumption dataset containing vehicle characteristics such as:

- Engine size
- Number of cylinders
- Fuel consumption
- CO₂ emissions

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow

1. Import libraries
2. Load and explore dataset
3. Data visualization
4. Feature selection
5. Train a Linear Regression model
6. Evaluate model performance

## Model

The project uses **Simple Linear Regression** from Scikit-Learn to model the relationship between engine size and CO₂ emissions.

## Results

The analysis shows a strong positive relationship between engine size and CO₂ emissions.

Larger engines tend to produce higher CO₂ emissions.
